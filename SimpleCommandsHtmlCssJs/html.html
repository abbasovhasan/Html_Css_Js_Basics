HTML Bölməsi

HTML (HyperText Markup Language) veb səhifənin strukturunu təyin edən işarələmə dilidir.

<h1>Salam Dünya</h1>

SEO (Search Engine Optimization) saytın axtarış motorlarında üst sıralarda çıxması üçün optimallaşdırılmasıdır.

Semantic tag-lər (məsələn, <header>, <footer>, <article>) kontentin məqsədini aydın göstərən HTML elementləridir.

HTML5 HTML dilinin son versiyasıdır. Video, audio, canvas, semantic tag-lər kimi yeni xüsusiyyətlər əlavə olunub.
