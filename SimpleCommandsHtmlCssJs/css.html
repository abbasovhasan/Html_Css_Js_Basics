CSS Bölməsi

CSS (Cascading Style Sheets) veb səhifənin dizaynını və tərtibatını idarə edən dildir.

body { color: blue; }

Reset CSS brauzerlərin default stillərini sıfırlamaq üçün istifadə olunur ki, bütün brauzerlərdə eyni görünüş əldə olunsun.

Box Model hər bir elementin ətrafında margin, border, padding və content qatlarını təsvir edir.

Content → Padding → Border → Margin

Responsivlik saytın müxtəlif ekran ölçülərinə uyğun görünməsini təmin etməkdir.

Media Query CSS-in fərqli cihaz ölçülərinə görə xüsusi stillər tətbiq etməsinə imkan verir.

@media (max-width: 600px) { body { background: lightblue; } }
